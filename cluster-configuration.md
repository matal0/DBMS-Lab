# Cluster Configuration   
## Cluster Settings   
- identify SQL to query current cluster settings.   
- identify cluster configuration files.   
- increase setting max\_connections to 103 using SQL.   
- decrease setting max\_connections back to 100 without using SQL.   
   
## SSL   
- configure the DB cluster to use SSL e.g. using self-signed certificate and private CA.   
- connect to DB labdb as user pgbench using psql having ssl enforced.
